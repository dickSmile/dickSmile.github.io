什么是markdown

Markdown 的目标是实现「易读易写」
Markdown 的语法全由一些符号所组成，这些符号经过精挑细选，其作用一目了然。
Markdown 语法的目标是：成为一种适用于网络的书写语言。Markdown 不是想要取代 HTML，甚至也没有要和它相近，
它的语法种类很少，只对应 HTML 标记的一小部分。Markdown 的构想不是要使得 HTML 文档更容易书写。在我看来，
HTML 已经很容易写了。Markdown 的理念是，能让文档更容易读、写和随意改。
HTML 是一种发布的格式，Markdown 是一种书写的格式。就这样，Markdown 的格式语法只涵盖纯文本可以涵盖的范围。

1.标题设置（让字体变大，和word的标题意思一样）

在Markdown当中设置标题，有两种方式：
第一种：通过在文字下方添加“=”和“-”，他们分别表示一级标题和二级标题。
第二种：在文字开头加上 “#”，通过“#”数量表示几级标题。（一共只有1~6级标题，1级标题字体最大）

代码：

# This is an H1 
## This is an H2`
### This is an H3
#### This is an H4
##### This is an H5
###### This is an H6

显示效果：

This is an H1

This is an H2

This is an H3

This is an H4

This is an H5
This is an H6

代码：

This is an H1 
=====
This is an H2`
-----

显示效果：

This is an H1
=
This is an H2`
-
个人觉得第一种方法更简洁显目

2. 块注释（blockquote）

通过在文字开头添加“>”表示块注释。（当>和文字之间添加五个blank时，块注释的文字会有变化。

代码：

>     加上5个blank
> 惟楚有才                                                                                      
> 于斯为盛

显示效果：

加上5个blank

惟楚有才
于斯为盛

另外，块注释允许多层叠加，例如：

代码：

>>> * 请问 Markdwon 怎么用？ - 小白
>> * 自己看教程！ - 愤青
> * 教程在哪？ - 小白

显示效果

  • 请问 Markdwon 怎么用？ - 小白
  • 自己看教程！ - 愤青
  • 教程在哪？ - 小白

3. 斜体和粗体

斜体：将需要设置为斜体的文字两端使用1个“”或者“_”夹起来
粗体：将需要设置为斜体的文字两端使用2个“”或者“_”夹起来
代码：

*斜体*或_斜体helloWord_
**粗体**
***加粗斜体helloWord***
~~删除线~~
  • 斜体 或 斜体helloWord
  • 粗体
  • 加粗斜体helloWord
  • 删除线

4. 无序列表

  1. 列表 在文字开头添加(, +, and -)实现无序列表。但是要注意在(, +, and -)和文字之间需要添加空格。（建议：一个文档中只是用一种无序列表的表示方式）

    代码：

    - 无序列表项 一
- 无序列表项 二
- 无序列表项 三

    显示效果：

    • 无序列表项 一
    • 无序列表项 二
    • 无序列表项 三
    1. 排版

    要让列表看起来更漂亮，你可以把内容用固定的缩进整理好：

    代码:

    • Lorem ipsum dolor sit amet, consectetuer adipiscing elit.
      Aliquam hendrerit mi posuere lectus. Vestibulum enim wisi,
      viverra nec, fringilla in, laoreet vitae, risus.
    • Donec sit amet nisl. Aliquam semper ipsum sit amet velit.
      Suspendisse id sem consectetuer libero luctus adipiscing.

    显示效果：

    • Lorem ipsum dolor sit amet, consectetuer adipiscing elit.
      Aliquam hendrerit mi posuere lectus. Vestibulum enim wisi,
      viverra nec, fringilla in, laoreet vitae, risus.
    • Donec sit amet nisl. Aliquam semper ipsum sit amet velit.
      Suspendisse id sem consectetuer libero luctus adipiscing.

    5. 有序列表

    使用数字后面跟上句号。（还要有空格）

    代码：

    1. 有序列表项 一
2. 有序列表项 二
3. 有序列表项 三

    显示效果：

    1. 有序列表项 一
    2. 有序列表项 二
    3. 有序列表项 三

    这里它的排序是从第一位开始的，与后面的无关，例如：

    代码：

    1. 有序列表项 三
    2. 有序列表项 一
    3. 有序列表项 五

    显示效果：

    1. 有序列表项 三
    2. 有序列表项 一
    3. 有序列表项 五

6. 链接（Links）

Markdown中有两种方式，实现链接，分别为行内式和参考式两种形式。

不管是哪一种，链接文字都是用 [方括号] 来标记。

  1. 行内式 要建立一个行内式的链接，只要在方块括号后面紧接着圆括号并插入网址链接即可，如果你还想要加上链接的 title 文字，只要在网址后面，用双引号把 title 文字包起来即可

    代码：

    [构互动](http://edu.bgou.top/ "welcom")是个知识分享平台
 <http://edu.bgou.top> 简短链接方式

    显示效果：

    构互动是个知识分享平台

    http://edu.bgou.top 简短链接方式

  2. 参考式：
    参考式的链接是在链接文字的括号后面再接上另一个方括号，而在第二个方括号里面要填入用以辨识链接的标记：

    代码

    你可以在[baidu][1]或者[Google][2]中搜索[构互动][3]来找到我们的平台.
[1]: http://www.baidu.com/ '百度搜索'
[2]: http://google.com/    '谷歌搜索'
[3]: http://edu.bgou.top/   构互动

    显示效果：

你可以在[baidu][1]或者[Google][2]中搜索[构互动][3]来找到我们的平台.

[1]: http://www.baidu.com/ ‘百度搜索’
[2]: http://google.com/ ‘谷歌搜索’
[3]: http://edu.bgou.top/ ‘构互动’

链接内容定义的形式为：

 方括号（前面可以选择性地加上至多三个空格来缩进），里面输入链接文字
 接着一个冒号
 接着一个以上的空格或制表符
 接着链接的网址
 选择性地接着 title 内容，可以用单引号、双引号或是括弧包着  

 链接网址也可以用方括号包起来：

 链接辨别标签可以有字母、数字、空白和标点符号，但是并不区分大小写

 隐式链接标记功能让你可以省略指定链接标记，这种情形下，链接标记会视为等同于链接文字，要用隐式链接标记只要在链接文字后面加上一个空的方括号

代码：

你可以在[baidu][]或者[Google][]中搜索[构互动][]来找到我们的平台.

   [BAIDU]: <http://www.baidu.com/> '百度搜索'
   [GOOGLE]:  http://google.com/   "谷歌搜索"
   [构互动]:   http://edu.bgou.top/   (构互动)

显示效果：

你可以在[baidu][]或者Google中搜索构互动来找到我们的平台.

[BAIDU]: http://www.baidu.com/ ‘百度搜索’

例子中把这几种语法都体现了，其中，隐式链接标记功能比较好用，有点像参考链接的意思，易读易理解。

8. 代码（HTML中所谓的Code）

实现方式有两种：

  1. 简单文字出现一个代码框。使用<blockquote>。（不是单引号而是左上角的ESC下面~中的）
    如果要在代码区段内插入反引号，你可以用多个反引号来开启和结束代码区段：

    代码：

    Use the printf() function.
    There is a literal backtick (`) here.

    显示效果：

    Use the printf() function.

    There is a literal backtick (`) here.

  1. 大片文字需要实现代码框。使用Tab和四个空格。

代码：

#include <stdio.h>
   int main(void)
   {
       printf("Hello world\n");
   }

显示效果

#include <stdio.h>
  int main(void)
  {
      printf("Hello world\n");
  }

10. 脚注（footnote）

在需要添加注脚的文字后加上脚注名字[^注脚名字],称为加注。 然后在文本的任意位置(一般在最后)添加脚注，脚注前必须有对应的脚注名字。

注意：经测试注脚与注脚之间必须空一行，不然会失效。成功后会发现，即使你没有把注脚写在文末，经Markdown转换后，也会自动归类到文章的最后。

代码：

使用 Markdown[^1]可以效率的书写文档, 直接转换成 HTML[^2], 你可以使用 Leanote[^Le] 编辑器进行书写。
[^1]:Markdown是一种纯文本标记语言
[^2]:HyperText Markup Language 超文本标记语言
[^Le]:开源笔记平台，支持Markdown和笔记直接发为博文

显示效果：

我使用的是webstom^1进行编写，它的功能很强大，不过要装上一个专用markdown插件[^2]才能识别

11. 下划线

你可以在一行中用三个以上的星号、减号、底线来建立一个分隔线，行内不能有其他东西。你也可以在星号或是减号中间插入空格。
当前后都有段落时，请空出一行：

代码：

我是分割线上方

 ---

 我是分割线下方

显示效果

我是分割线上方

我是分割线下方

13. 公式

当你需要在编辑器中插入数学公式时，可以使用两个美元符 $$ 包裹 TeX 或 LaTeX 格式的数学公式来实现。提交后，问答和文章页会根据需要加载 Mathjax 对数学公式进行渲染。如：

代码：

$$\sum_{i=1}^n a_i=0$$
$$f(x_1,x_x,\ldots,x_n) = x_1^2 + x_2^2 + \cdots + x_n^2 $$
$$\sum^{j-1}_{k=0}{\widehat{\gamma}_{kj} z_k}$$

显示效果：

$$\sum_{i=1}^n a_i=0$$
$$f(x_1,x_x,\ldots,x_n) = x_1^2 + x_2^2 + \cdots + xn^2 $$
$$\sum^{j-1}{k=0}{\widehat{\gamma}_{kj} z_k}$$

14. 转义符

如果你的描述中需要用到 markdown 的符号，比如 # * 等，但又不想它被转义，这时候可以在这些符号前加反斜杠，如 \ # * 进行避免。

代码：

\_不想这里的文本变斜体\_
\*\*不想这里的文本被加粗\*\*

显示效果：

_不想这里的文本变斜体_
**不想这里的文本被加粗**

15. 流程图和图标

  1. 图表： 不管是哪种方式，第一行为表头，第二行分隔表头和主体部分，第三行开始每一行为一个表格行。
    列于列之间用管道符|隔开。原生方式的表格每一行的两边也要有管道符。
    第二行还可以为不同的列指定对齐方向。默认为左对齐，在-右边加上:就右对齐。

代码：

产品|价格
-|-:
Leanote 高级账号|60元/年
Leanote 超级账号|120元/年

显示效果：

产品 价格
Leanote 高级账号 60元/年
Leanote 超级账号 120元/年
